### Developed for Martin Luther King High School (Detroit, MI)  
**By Xavier Honablue, M.Ed | CognitiveCloud.ai**

---

## 🧠 Overview

**Crusader Function Lab** is an interactive Streamlit application designed to help students explore mathematical functions through **real-world Detroit-based contexts**.

This lesson transforms abstract math into something students can *see, feel, and connect to* — from DDOT bus routes to Detroit factory wages.

---

## 🎯 Learning Objectives

Students will:
- Understand what a function is
- Identify different types of functions (linear, exponential, quadratic, logarithmic)
- Connect math to real-world scenarios
- Interpret graphs and relationships
